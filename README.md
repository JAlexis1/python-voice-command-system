# python-voice-command-system
Este proyecto corresponde a un laboratorio práctico donde se implementa un sistema básico de reconocimiento de voz utilizando Python.

El sistema permite:
- Grabar audio desde el micrófono.
- Convertir voz a texto usando la API de Google Speech Recognition.
- Detectar palabras clave.
- Ejecutar comandos simples según la frase reconocida.

## Objetivos del laboratorio

- Configurar un entorno de trabajo en Python para procesamiento de audio.
- Utilizar la librería `SpeechRecognition` para convertir voz en texto.
- Grabar audio desde el micrófono y almacenarlo temporalmente.
- Implementar un sistema básico de comandos por voz.
- Extender el código base agregando nuevas funcionalidades.
